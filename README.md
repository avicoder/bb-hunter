## Installation

This package is distributed via npm:

```
npm install bb-hunter
```


## The Library

### import

```
const bb-hunter = require('bb-hunter');
```

### all

```
bb-hunter.all
```

Return an array of names



### random

```
bb-hunter.random()
```

Return an random of name of bug bounty hunter
